## 📌 About

**Diep.io WASM Patch**  
Created by **tinysweet** *(aka **DDatiOS**, old name)*

This project focuses on **reverse engineering and patching the Diep.io WebAssembly (WASM)** binary to modify internal behaviors such as **FOV, logic handling, or other client-side mechanics**.

> ⚠️ **Educational & research purpose only.**  
> This repository is intended for learning **WebAssembly reversing**, **binary patching**, and **browser internals**.

---

## ✨ Features

- 🔧 Patch internal WASM logic
- 🔍 Analyze Diep.io WebAssembly structure
- 🧠 Reverse engineering–friendly workflow
- 🖥️ Browser-based (Tampermonkey / Local WASM loading)
- 🧪 Tested on latest Diep.io builds *(as of patch date)*

---

## 🛠️ Tech Stack

- **JavaScript**
- **WebAssembly (WASM)**
- **Tampermonkey**
- **IDA Pro / wasm2wat / custom tools**

---

## 📂 Project Scope

- WASM binary analysis  
- Opcode / function patching  
- Address relocation handling  
- Runtime injection & loading  

---

## 🚀 Usage

1. Dump or load the original `diep.wasm`
2. Apply patch logic (manual or scripted)
3. Load patched WASM via browser / userscript
4. Verify behavior in-game

> Detailed usage depends on the patch version and method used.
